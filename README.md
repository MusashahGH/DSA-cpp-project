# Linked List Implementation (C++)

A menu-driven singly linked list program built in C++ as part of my DSA coursework.

## Features
- Insert, Display, Search, and Delete operations
- Supports string data (preserves leading zeros)
- Dynamic memory allocation using pointers
- Demonstrates algorithm properties: Definiteness, Finiteness, Effectiveness

## How to Run
g++ linked_list_string_cpp.cpp -o output
./output

## Concepts Used
- Singly Linked List
- Dynamic Memory Management (new / delete)
- OOP in C++ (Classes, Encapsulation)
