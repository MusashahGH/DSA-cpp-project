# DSA-cpp-project
# 🔗 Linked List Implementation (C++)

A menu-driven singly linked list program built in C++ as part of my DSA coursework.

---

## ✨ Features

- ➕ **Insert** nodes with string data (preserves leading zeros)
- 📋 **Display** all elements in the list
- 🔍 **Search** for any value in the list
- 🗑️ **Delete** nodes by value
- ⚙️ Demonstrates algorithm properties: Definiteness, Finiteness, Effectiveness

---

## 🖥️ How to Run

### Prerequisites
- C++ Compiler (g++ recommended)
- Terminal / Command Prompt

### Steps
```bash
# 1. Compile the program
g++ "PROJECT [LINKED LIST CONECT MANAGEMENT].cpp" -o linkedlist

# 2. Run the executable
./linkedlist

# For Windows
linkedlist.exe
```
